The Four Layers of TCP/IP
Understand the parts of TCP/IP and the roles of each layer of technology.

TCP/IP is a suite of technologies that make up the underlying networking technology of the internet. The name stands for Transmission Control Protocol and Internet Protocol, but itâ€™s really a term for a group of technologies. There are four layers to this technology:

  • Application layer
    • This is where protocols like HTTP (for serving webpages) and SMTP (for emails) live
  • Transport layer
    • This is how the transmission of data is governed
    • TCP is here
  • Internet layer
    • Where IP and DNS technologies make it possible to specify where packets are going
  • Network layer
    • The layer of which the physical communication of data takes place
